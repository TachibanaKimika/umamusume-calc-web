部分数据来源: [wrrwrr111/pretty-derby](https://github.com/wrrwrr111/pretty-derby)


## 使用说明

### **练习模拟**

使用6张支援卡, 可以模拟单次或若干次练习次数最后生成盒须图查看结果的数值以及分布情况.

### **单卡分析**

对比支援卡数据, 并且可以对任意练习进行单次模拟比较数据.

### **数据统计**

统计支援卡的属性并将其可视化.
